# 3. システムアーキテクチャ

## 3.1 レイヤードアーキテクチャ
```
┌─────────────────────────────────────────────┐
│          Presentation Layer (React)          │
│  - Components (Menu, HUD, Settings, Editor)  │
└─────────────────────────────────────────────┘
                        ↕
┌─────────────────────────────────────────────┐
│         Application Layer (Services)         │
│  - Game Service, Storage Service, Audio      │
└─────────────────────────────────────────────┘
                        ↕
┌─────────────────────────────────────────────┐
│          Domain Layer (Game Core)            │
│  - Entities, Physics, Rules, State           │
└─────────────────────────────────────────────┘
                        ↕
┌─────────────────────────────────────────────┐
│       Infrastructure Layer (Canvas)          │
│  - Rendering, Input, Asset Loading           │
└─────────────────────────────────────────────┘
```

## 3.2 コンポーネント構成
```
App
├── GameContainer (React)
│   ├── MainMenu
│   ├── GameUI
│   │   ├── HUD
│   │   ├── PauseMenu
│   │   └── GameOverScreen
│   ├── SettingsPanel
│   └── LevelEditor
│
└── GameCanvas (Canvas)
    ├── GameEngine
    │   ├── GameLoop
    │   ├── StateManager
    │   └── EventBus
    ├── EntityManager
    │   ├── Ball
    │   ├── Paddle
    │   ├── Blocks
    │   └── PowerUps
    ├── Systems
    │   ├── PhysicsSystem
    │   ├── CollisionSystem
    │   ├── RenderingSystem
    │   └── ParticleSystem
    └── AssetManager
```
